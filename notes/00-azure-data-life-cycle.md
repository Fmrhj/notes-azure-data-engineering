# Data Life Cycle in Azure

Ingestion:

- Data Factory
- HDFS
- PolyBase

Explore:

- SQL Data Warehouse (Polybase)
- Databricks

Preparation and Train:

- Data Factory
- Databricks
- SQL Data Warehouse: Petabyte oriented analysis of data

Model & Serve:

- Azure SQL
- Data Warehouse
- Streaming Analytics

Store:

- Blob Storages
- Data Lake v2
- Cosmos DB (non-relational, wide-column)
- Azure SQL

Consumers:

- BI + Reporting
- Advanced analytics
- Real Time Analytics

Sources:

- Images
- Structured or non-structured data
- Video
- Many more
